ゲームしながらプログラミングできそうなの

Feb 10th, 2015 5:46 pm

ゲームしながらプログラミング出来そうなもの個人的なまとめ
他に面白そうなのあったら教えて

  • Minecraft
    • ComputerCraft
      • 使用言語: Lua
      • マイクラにコンピュータを追加してプログラムを書いて自動化したりする感じ
  • HackforPlay
    • 使用言語: Javascript
    • Enchant.jsを使ってコードを改造する感じ
  • CodinGame
    • 使用言語: たくさん
    • ゲームがたくさん
  • RoboCode
    • 使用言語: Java もしくは .NET
    • ロボットを作って戦わせる
  • CheckiO
    • 使用言語: Python
    • ID登録の時点でプログラムを書かなければならない
